## ü§ù Getting Help

### Quick Help
1. **Check the Documentation**: [docs.ucli.tools](https://docs.ucli.tools)
2. **Search Existing Issues**: Use GitHub search across all repositories
3. **Ask in Discussions**: Post in [GitHub Discussions](https://github.com/ucli-tools/community/discussions)

### Before Asking
- **Search first**: Check if your question has been asked before
- **Include details**: OS, tool versions, error messages
- **Be specific**: Clear problem description and steps to reproduce

### Response Times
- **Community**: Usually within 24-48 hours
- **Maintainers**: Usually within 12-24 hours for critical issues
- **Complex issues**: May take longer for investigation
