    Contact & Community¶

    Get in touch with the UCLI Tools community! We're here to help, collaborate, and build amazing tools together.

    üåü Community Resources¶

    GitHub Organization¶

    • Organization: ucli-tools
    • Main Repository: ucli
    • Documentation: docs.ucli.tools
    • Installation: install.ucli.tools

    Communication Channels¶

    üí¨ GitHub Discussions¶

    Our primary community forum for: - General questions and help - Feature requests and ideas - Show and tell - General discussion

    üêõ GitHub Issues¶

    For bug reports and technical issues: - ucli issues - gits issues - mdtexpdf issues - mdaudiobook issues

    ü§ù Getting Help¶

    Quick Help¶

    1. Check the Documentation: docs.ucli.tools
    2. Search Existing Issues: Use GitHub search across all repositories
    3. Ask in Discussions: Post in GitHub Discussions

    Before Asking¶

    • Search first: Check if your question has been asked before
    • Include details: OS, tool versions, error messages
    • Be specific: Clear problem description and steps to reproduce

    Response Times¶

    • Community: Usually within 24-48 hours
    • Maintainers: Usually within 12-24 hours for critical issues
    • Complex issues: May take longer for investigation

    üë• Community Guidelines¶

    Be Respectful¶

    • Treat everyone with respect and kindness
    • Use inclusive language
    • Focus on constructive feedback
    • Assume good intent

    Be Helpful¶

    • Share knowledge and solutions
    • Help newcomers get started
    • Contribute back to the community
    • Participate in discussions

    Stay On Topic¶

    • Keep discussions relevant to UCLI Tools
    • Use appropriate channels for different topics
    • Respect thread topics
